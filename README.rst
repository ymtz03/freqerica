freqerica
========================

Install
---------------

.. code-block:: bash

   git clone https://github.com/ymtz03/freqerica.git
   cd freqerica
   python setup.py install
   
Test
---------------
   
.. code-block:: bash

   python setup.py test

Example
---------------
   
.. code-block:: bash

   cd example/LiH
   python LiH.py
   open result.html
